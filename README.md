Exploratory Data Analysis (EDA)

Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce dataset. The objective is to identify patterns, trends, outliers, and relationships within the data using statistical analysis and visualizations.

Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

Dataset Features

* OrderID
* Date
* CustomerID
* Product
* Quantity
* UnitPrice
* PaymentMethod
* OrderStatus
* ItemsInCart
* TotalPrice

Analysis Performed

* Data Inspection
* Missing Value Analysis
* Descriptive Statistics
* Product Distribution Analysis
* Payment Method Analysis
* Order Status Analysis
* Histogram Visualization
* Outlier Detection using Boxplot
* Correlation Analysis using Heatmap

Key Findings

* Average Order Value: ₹1053.97
* Median Order Value: ₹823.62
* Printer was the most frequently ordered product.
* Online Payment was the most preferred payment method.
* Several high-value outliers were identified.
* Strong positive correlation was observed between UnitPrice and TotalPrice.
